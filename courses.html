Introduction to Computer Science

An overview of fundamental concepts in computer science, including algorithms, data structures, and problem-solving. This course serves as a foundation for further studies in the field.

Web Development Technologies

Explore the technologies and languages used in modern web development. Topics include HTML, CSS, JavaScript, and frameworks like React and Angular.

Database Design and Management

Learn the principles of designing and managing databases. Topics cover relational database systems, SQL, normalization, and database optimization.

Programming

Delve into the exciting world of programming and machine learning. This course covers algorithms, statistical modeling, and the practical application of machine learning in various domains.
